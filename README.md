# Lead Predictor 🚀

Lead Predictor is an interactive web-based forecasting tool designed to help sales and marketing professionals calculate the exact number of prospects and leads needed to hit their revenue goals. 

By adjusting dynamic conversion rates, users can instantly forecast requirements, optimize their sales funnel, and learn how to implement strategic surveys to maximize profits.

## 🌟 Key Features

- **Dynamic Funnel Forecasting:** Instantly calculate how many Prospects and Leads you need to secure your target Customers.
- **Interactive Adjustments:** Use sleek range sliders to adjust your *Lead Response Rate* and *Prospect Response Rate* and watch the numbers recalculate in real-time.
- **Visual Data Integration:** Features an overlapped, horizontal bar chart powered by Chart.js to visualize the sales funnel pipeline over a chronological layout.
- **Educational Survey Insights:** Includes a dedicated panel explaining 4 core customer preference surveys (CSAT, NPS, CES, PMF), complete with ideal timing and scoring logic.
- **Bilingual Support:** Full internationalization (i18n) supporting both **English** (EN) and **Bulgarian** (BG) languages out of the box.
- **Modern SaaS UI:** Clean, responsive, and action-oriented interface with rich typography (Inter font) and smooth micro-interactions.

## 🛠️ Tech Stack

- **HTML5:** Semantic structure and layout.
- **CSS3:** Modern CSS with custom properties (CSS variables), Flexbox, CSS Grid, and responsive media queries.
- **Vanilla JavaScript:** Logic, event listeners, calculations, and real-time DOM manipulation. No heavy frameworks!
- **Chart.js (v4):** Used via CDN for rendering the responsive funnel chart visualization.

## 🚀 Getting Started

Since this is a client-side vanilla web application, there is no complex build process or installation required!

1. **Clone the repository:**
   ```bash
   git clone https://github.com/your-username/lead-predictor.git
   ```
2. **Open the project:**
   Simply double-click the `index.html` file to open it in your default web browser.

## 💡 How to Use

1. **Set your targets:** Enter your *Total Revenue* goal and *Avg. Order Value* in the left sidebar.
2. **Adjust Rates:** Use the sliders in the bottom section to simulate improvements to your Lead and Prospect response rates.
3. **Analyze:** Watch the core metrics (Top Stats) and the visual funnel chart (Main Panel) update dynamically based on your parameters.
4. **Learn & Optimize:** Read through the *Survey strategies* section to see how CSAT or NPS can be structurally applied to improve the very conversion rates you are forecasting.
